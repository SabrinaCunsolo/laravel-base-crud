Oggi pomeriggio esercitatevi con le migrations:
1. create un nuovo database con phpMyAdmin
2. create un nuovo progetto Laravel
3. create una nuova migration per definire la struttura di una tabella products
4. tramite phpMyAdmin inserite qualche riga nella tabella
5. create un model Product
6. create un ProductController usando l'opzione --resource affinché Laravel predisponga già tutte le funzioni per le CRUD
7. create le rotte resource
8. create una view index per listare tutti i prodotti presenti nel db -> per ogni prodotto aggiungete un link alla relativa pagina di dettaglio
9. create una view show per visualizzare i dettagli di un singolo prodotto
